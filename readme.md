Backend setup:

    ```
    cd server
    npm install

    ```

Run backend:

    ```
    node index.js

    ```
    

Frontend setup:

    ```
    cd client
    npm install

    ```

Create a .env file in /client/

    ```
    BACKEND_URL=ws://localhost:8600

    ```
        
Run Frontend:

    ```
    npm run dev
    ```
